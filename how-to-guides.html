How to Use vClock - Complete Guide

Using the Clock

  • The main clock displays your current local time
  • Use the timezone dropdown to view time in different regions
  • The date is displayed below the clock in your selected timezone

Stopwatch Features

  • Click 'Start' to begin the stopwatch
  • Use 'Stop' to pause the current time
  • The 'Reset' button returns the stopwatch to 00:00:00

Setting Alarms

  • Select your desired alarm time using the time picker
  • Click 'Set Alarm' to activate
  • Use 'Test Sound' to verify your audio settings
  • 'Cancel All Alarms' will clear your alarm list

Pro Tips

  • Bookmark vClock for quick access
  • Use keyboard shortcuts for faster control
  • Keep the tab open for alarm functionality
